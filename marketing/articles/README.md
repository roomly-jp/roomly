# 記事作成プロジェクト

Roomly用のブログ記事雛形をまとめたディレクトリです。

各ファイルはタイトルと簡単な構成が記載されたテンプレートになっており、
コピーや修正をして実際の記事として使うことができます。

**作業手順**
1. 該当するテンプレート（例 `01_transparent_pricing.md`）を開く。
2. タイトルと導入文を必要に応じて編集。
3. セクションの箇条書きを肉付けし、本文を追加。
4. 画像や引用、リンクを貼る場合はMarkdownで追記。
5. 完成したら`public/blog`など実際のブログディレクトリにコピー。

---

このディレクトリはLunaposプロジェクトで使っていたワークフローを踏襲しています。
同様に、WordPressやGatsbyなどコンテンツ管理システムに流用可能です。
