**Roomlyの対応方針**

これらはすべて「必要になれば追加料金」の形態で提供されており、小規模事業者にとっては予想外のランニングコストになるリスクがある。Roomlyでは以下の方針で差別化を図る予定。

1. **コア機能を無制限提供** – 例えばデータバックアップやAPI連携はすべて基本料金内（区画数のみ課金）。
2. **一部機能はモジュール化して月額オプション** – マイページやオーナーページ、SMS送信など、使用率が低いが価値のある機能はオプションとして用意（価格は廉価設定）。
3. **初期費用ゼロ** – どの機能を選んでも初期導入料は発生せず、トライアル後も継続的に追加可能。
4. **将来的には機能バンドル** – ある規模以上（例 500区画〜）ではオプションをすべて含む「プロフェッショナルパック」を用意し、価格上昇分を透明化。

この方針により、「オプション漬けで請求額が高くなる」ReDocSとは対照的なシンプルさを打ち出す。

##### オプション込みの想定コスト
ReDocSを導入する一般的な管理会社が、データサポート・銀行連携・入居者/オーナーページ・メール送信・ストレージなど主要オプションをすべて契約した場合、月額は約¥10,000を超え、初年度のトータルコストは¥300K前後になる試算だ。これに対しRoomlyは同機能すべてを追加料金なしで提供するため、実質的な差はさらに広がる。

### 3-2. なぜこの価格設定か

**10区画まで（¥0/月）の根拠:**
- 個人オーナーの自主管理なら10区画で十分（アパート1〜2棟相当）
- 管理会社でも「まず1物件試してみる」が可能な規模
- ReDocSの30日トライアルより遥かに長い検討期間を提供
- 無料枠が広すぎると有料転換率が下がるため、実用的かつコンパクトに設定

**50区画まで（¥5,000/月・税込）の根拠:**
- 個人〜小規模管理会社の主力プラン（アパート5〜10棟相当）
- ReDocSライト（¥3,278税抜≒¥3,606税込）と同価格帯だが、オプション込みでRoomly優位
- 月¥5,000は「経費として気にならない」心理的ライン
- 年間¥60,000。ReDocS初期費用¥55K + 月額を考えると初年度から優位

**100区画まで（¥10,000/月・税込）の根拠:**
- ReDocSライト（¥3,278税抜≒¥3,606税込）に主要オプションを全て付けた場合、月額は¥10,000前後になる見込み
- Roomlyは同機能をすべて標準提供するため、追加費用なしで利用可能
- 年間トータルコストは¥120,000。ReDocSと比較して初期費用込みでも遜色ない（オプション課金が発生するとRoomly優位）
- ¥10,000は「部門予算+α」で通しやすい心理的ライン
- 中規模管理会社の主力プラン

**300区画まで（¥15,000/月・税込）の根拠:**
- ReDocSミドル（推定¥7,678税抜≒¥8,446税込）に多数のオプションを追加すると月額は¥15,000超になる想定
- Roomlyでは同レベルの機能を標準提供し、ライセンスだけで運用可能
- 年間¥180,000。リスクフリーでオプション追加の誤算がない点で優位
- 「3万前後の部門予算」で堂々と導入できる

**500区画まで（¥20,000/月・税込）の根拠:**
- ReDocSミドル相当の機能にすべてオプションを付けると月額¥20,000前後に膨らむ
- 中規模管理会社向けの本格プランとして価格帯を設定
- 年間¥240,000。区画数と機能での付加価値を合わせるとコストパフォーマンス良

**1,000区画まで（¥25,000/月・税込）の根拠:**
- ReDocSのアドバンスプラン想定価格に加え、オプションを全て含めた場合のコストを上回る
- 年間¥300,000。複数拠点運用や専任サポートが付加される層
- 1日あたり833円。大企業向けプロフェッショナル価格帯

**2,000区画まで（¥30,000/月・税込）の根拠:**
- 年間¥360,000。ReDocS推定見積に対し、オプション込みでほぼ同等か若干高いが、初期費用ゼロが差別化
- 2,000区画以上はカスタム見積が必要なため、セルフサービス型の最終到達点
- エンタープライズ向けの最上位セルフプラン

## 4. スケーリング戦略（営業不要型）

### 結論: **10区画 → 50区画 → 100区画 → 300区画 → 500区画 → 1,000区画 → 2,000区画 → カスタム まで、すべてセルフサービス型で対応**

```
10区画まで（¥0）
  ↓ 事業成長
50区画まで（¥5,000/月）★小規模フェーズ
  ↓ さらに成長
100区画まで（¥10,000/月）★成長フェーズ
  ↓ さらに成長
300区画まで（¥15,000/月）★安定フェーズ
  ↓ 企業拡大
500区画まで（¥20,000/月）★中規模フェーズ
  ↓ スケール拡大
1,000区画まで（¥25,000/月）★エンタープライズ予備軍
  ↓ 大型案件
2,000区画まで（¥30,000/月）★ボリュームフェーズ
  ↓ 超大規模
カスタム見積→要相談

全ステージで営業・導入チーム不要。API・自動化で対応
```

### 4-1. なぜ営業が不要か

| 要素 | 従来型（営業必要） | Roomly型（営業不要） |
|------|------|------|
| 契約 | RFP → 売上管理 → デモ → 契約 | Webサイト → クレジットカード決済で即導入 |
| 導入 | 専任が現地対応・教育 | ナレッジベース・チュートリアル動画で自習 |
| カスタマイズ | 営業が要望聞き取り → 仕様書化 | API/Webhook で外部ツール連携 |
| 支援 | CSM がモニタリング | ダッシュボード・自動通知で顧客セルフサービス |
| スケール判定 | 営業が「アップグレード案内」 | 使用量が上限に達したら自動アラート（顧客が自分で判断） |

### 4-2. 区画数上限でのシームレスな移行

1. **データ移行なし** → すべての区画数レベルが同一DB（上限のみ異なる）
2. **機能差なし** → スケールしても必要な機能がいきなり消える不安がない
3. **API無制限** → 500区画以上なら外部ツール連携で機能拡張可能
4. **自動スケーリング** → 区画数が増えても人的対応なしで利用継続

### 4-3. 実装フェーズ

```
Phase 1（0-12ヶ月）: ~500区画 中心
  ├── 小規模管理会社 50-100社獲得
  ├── プロダクト構成最適化（UXを磨く）
  └── ベータパートナー事例を作成

Phase 2（12-24ヶ月）: ~2,000区画 追加・API開発
  ├── 中規模管理会社へのリーチ
  ├── REST API v1 公開
  ├── Webhook 実装
  └── 合計実績 150-200社を目指す

Phase 3（24-36ヶ月）: ~10,000区画 フェーズ投入
  ├── API v2 強化
  ├── マルチテナント管理改善
  ├── パフォーマンス最適化
  └── 年間 MRR ¥500万超を目指す

Phase 4（36ヶ月以降）: 10,000超 カスタム対応
  ├── 特殊要件は「相談ベース」（エンジニアが対応）
  ├── ただしコア機能への組み込みは避ける
  └── 「外部連携で実現可能か」を最優先に検討
```

**Phase 1-2は個人でもスケール可能（Lunaposと同じモデル）。営業チーム不要。**

## 10. まとめ

### ビジョン

**「中小企業が大企業に成長する過程を、同じプロダクトで全面支援」**

- 10区画で始まったユーザーが、10年後に2,000区画企業に成長してもRoomlyを使い続けられる
- 成長に応じた料金体系（10区画無料 → 50区画¥5,000/月 → 100区画¥10,000/月 → 300区画¥15,000/月 → 500区画¥20,000/月 → 1,000区画¥25,000/月 → 2,000区画¥30,000/月 → 2,000超カスタム見積）。主要オプションはすべて標準搭載。
- 営業が不要なセルフサービス型で、スケール時の面倒な切り替えなし
- **ReDocsより高めの月額だが、初期費用ゼロ＆オプション込みで1年目TCOは優位**

### 戦略ポイント

1. **営業チーム不要** = 低コスト体質が実現
   - 導入まで顧客セルフサービス
   - API連携で独自要件に対応
   - スケール判定も顧客が自分で判断

2. **競争力ある価格と優れた初期費用体系**
   - 10区画：¥0/月 → 50区画：¥5,000/月 → 100区画：¥10,000/月 → 300区画：¥15,000/月 → 500区画：¥20,000/月 → 1,000区画：¥25,000/月 → 2,000区画：¥30,000/月 → 2,000超：カスタム見積
   - 上記には入居者/オーナー向けマイページ、メール/SMS送信、データバックアップなど主要オプションをすべて含む
   - ReDocsより高い月額だが、初期費用ゼロ＆オプション課金のリスクがないため1年目TCOでは依然優位
   - 成長フェーズごとに選択肢がある
   - 一度導入したら「乗り替え」の手間がない

3. **プラン名なし、シンプルな「区画数×価格」のみ**
   - プラン名による機能差の誤解がなくなる
   - 区画数が増える = 自動的に機能が増える（税込価格が上がる）と理解しやすい
   - 小規模の2〜3人社員の管理会社も、2,000区画企業も同じUIで運用可能

### 最優先アクション

1. **プロダクト完成** → ベータ版を実用レベルに（10区画無料 + 50区画・100区画プラン）
2. **ベータパートナー1社確保** → 実績ゼロを脱却・事例化
3. **比較サイト掲載** → SEO経由のリード獲得開始
4. **API設計ドキュメント** → 2,000区画プラン追加時に公開準備開始
